# 📚 Library Management System (Java + JDBC + MySQL)

A console-based Library Management System developed using Java, JDBC, and MySQL to manage library operations efficiently through a menu-driven interface.


## 🚀 Features

- Add new books with ID, title, and author  
- View all available books  
- Issue books to users  
- Return books with status update  
- Delete books from system  
- Persistent data storage using MySQL database  


## 🧠 Core Functionalities

- Menu-driven CLI system  
- JDBC-based database connectivity  
- Object-Oriented design (Book model, DAO layer)  
- Exception handling for database operations  


## 🛠 Technologies Used

- Java  
- JDBC (Java Database Connectivity)  
- MySQL  
- Scanner (CLI Input Handling)
